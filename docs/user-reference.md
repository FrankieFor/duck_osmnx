# User Reference

This is the User Reference for the ducknx package. If you are looking for an introduction to ducknx, read the [Getting Started](getting-started.md) guide. This guide describes the usage of ducknx's public API.

ducknx 2.0 is released: read the [migration guide](https://github.com/gboeing/osmnx/issues/1123).
